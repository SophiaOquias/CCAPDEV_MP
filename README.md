## Installation Instructions 
To run locally: 
1. Install all dependencies in `package.json` 
```shell
npm install
```
2. Run the server using the script defined as 
```shell
npm run dev
```
3. Open [http://localhost:3000](http://localhost:3000) on your browser to open the app
